# Política de Privacidade  
Aplicativo: Lista de Tarefas

## Introdução  
O aplicativo **Lista de Tarefas** foi desenvolvido com foco na organização pessoal e na privacidade do usuário. Esta Política de Privacidade explica como os dados são tratados dentro do aplicativo e esclarece que nenhuma informação pessoal é coletada ou enviada para servidores externos.

## Coleta e Uso de Informações  
O **Lista de Tarefas** **não coleta, armazena, transmite ou processa dados pessoais** dos usuários.  
Todas as informações inseridas — como tarefas, lembretes e anotações — são utilizadas exclusivamente para o funcionamento local do aplicativo.

## Armazenamento dos Dados  
Os dados das tarefas são armazenados **localmente no dispositivo do usuário**, utilizando a tecnologia **Room (SQLite)**.  
- Nenhuma informação é enviada para servidores externos.  
- Nenhuma informação é sincronizada com a nuvem.  
- Nenhum dado é compartilhado com terceiros.

O usuário pode excluir suas tarefas a qualquer momento diretamente no aplicativo.

## Publicidade  
Durante a fase de testes, o aplicativo utiliza **AdMob em modo de teste**, que não coleta dados reais de usuários.  
Após a publicação oficial, o aplicativo poderá exibir anúncios por meio do **Google AdMob**, que pode coletar dados técnicos mínimos necessários para exibição de publicidade, conforme as políticas do Google.  
O aplicativo poderá oferecer uma opção para **remover anúncios**.

## Público-Alvo  
O aplicativo é destinado a **adolescentes e adultos**.  
Não é projetado para uso por crianças.

## Segurança  
Como os dados são armazenados apenas no dispositivo do usuário, a segurança depende das proteções do próprio sistema operacional Android.  
O aplicativo não acessa, transmite ou processa informações sensíveis.

## Compartilhamento de Dados  
O **Lista de Tarefas** **não compartilha** dados com terceiros, exceto pelas informações técnicas mínimas tratadas automaticamente pelo AdMob (quando ativado), conforme as políticas do Google.

## Retenção e Exclusão  
Todos os dados permanecem no dispositivo do usuário até que ele decida excluí-los manualmente.  
O aplicativo não mantém cópias externas e não possui acesso aos dados armazenados.
